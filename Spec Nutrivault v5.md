Démarrons la verion 5.0 de Nutrivault !

Veuillez organiser ces fonctionnalités en sprints de développement, en tenant compte des dépendances et de la complexité.

## Feature Requests

### Manage RBAC

- As an admin, I want to create and manage user roles (e.g., admin, practitioner, nurse).
- As an admin, I want to assign permissions to roles for accessing different parts of the application
- As an admin, I want to assign users to roles.
- As a practitioner, I want to view only the sections of the application that my role permits.

### UI update

- Enlever la date de naissance dans la vue patient
- dans les custom fields, rajouter la capacité de l'affichier dans la vue de tout les patients (en plus de la vue detaillée du patient et de la visite)
- Rajouter un type de custom field 'calculé' qui permet de faire des opérations simples (+,-,*,/) entre d'autres custom fields. Par exemple IMC = poids / (taille * taille), ou l'age = date du jour - date de naissance

### Measures Tracking

I want to be able to manage measure that I want to follow for my patients. 
    - As a admin, I want to define custom measures (e.g., weight, blood pressure) for each patient.
    - As a practitioner, I want to log measure values with timestamps.
    - As a practitioner, I want to visualize measure trends over time with graphs.
    - A measre can be linked to a visit
    - A measure can be linked to a patient directly
    - A measure can be either a value, a text, a boolean or a math expression (e.g., BMI calculated from weight and height)
    - A measure can have normal ranges defined for different demographics (e.g., age, gender)
    - A measure can have alerts defined for out-of-range values.

### billing and email templates

- As a admin, I want to create and manage billing templates for different services.
- As a admin, I want to create and manage email templates for patient communications.
- As a practitioner, I want to customize invoice templates with my branding and ly signature.

### Communication to patients

- As a practitioner, I want to send automated appointment reminders to patients via email.
- As a practitioner, I want to send follow-up emails after appointments with personalized content including conclusion generated by an AI based on the consultation notes.

### Data visualization

- As a practitioner, I want to see dashboards with key metrics about my patients' health trends.
- As a admin, I want to generate reports on billing and communication effectiveness.
- As a practitioner, I want to see dashboards with key financial metrics.
